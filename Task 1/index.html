PC parts

A personal computer (PC) is a multi-purpose microcomputer whose size, capabilities, and price make it feasible for individual use. Personal computers are intended to be operated directly by an end user, rather than by a computer expert or technician.
In order to build a PC We need to know its hardware parts. You can visit techtarget to know more about pc parts and more information.

You can also know more about PCs by visiting Wikipedia

We can summarize the main PC hardware parts into six main components

  • Motherboard
  • Central Processing Unit (CPU)
  • Random Access Memory (RAM)
  • Graphics Processing Unit (GPU)
  • Power Supply Unit (PSU)
  • Hard Drives
